# Наоми для iOS

Нативный iPhone-клиент для [Наоми](https://github.com/PopaSpinka/naomi-agent_3) — личного
ИИ-компаньона, который живёт на твоём Маке. Это тот же самый разговор, что в вебе и
телеграме: одна история, один характер, один дом — теперь и в кармане.

Как и сама Наоми, проект писался для одной конкретной семьи и публикуется как есть.
Маленький, честный, хорошо прокомментированный (по-русски) — бери и перекраивай под себя.

## Что умеет

- **Один разговор со всеми каналами.** При запуске подтягивает общую историю — видно всё,
  что писалось из веба и телеграма. Отправил с айфона — увидишь на Маке, и наоборот.
- **Живой ответ.** Наоми отвечает по буквам, как в вебе — тот же поток `/api/chat`.
  Когда она что-то делает руками (кондиционер, телевизор), в пузыре видны плашки действий.
- **Интерфейс как у людей.** Тёплая палитра веба, тёмная тема, стеклянное поле ввода
  с системным размытием края (iOS 26), сообщения просвечивают сквозь клавиатуру —
  как в iMessage. Клавиатура не дёргает чат, если листаешь старую переписку.
- **Без сборщиков и зависимостей.** Чистый SwiftUI + URLSession, ни одного стороннего
  пакета — философия Наоми и здесь.

## Как собрать

Понадобится Mac с Xcode 26+ и работающая Наоми (см. основной репозиторий).

1. **Открой сервер Наоми домашней сети** — в `backend/.env` основного проекта добавь
   `NAOMI_HOST=0.0.0.0` и перезапусти. Наружу в интернет ничего не торчит:
   граница — пароль твоего Wi-Fi.
2. **Открой `Naomi.xcodeproj`**, в Signing & Capabilities выбери свой Team
   (хватает бесплатного Apple ID), подключи айфон и жми Run.
3. **Впиши адрес Мака** в настройках приложения (шестерёнка сверху):
   `http://имя-мака.local:8787`. Имя видно в macOS: Настройки → Основные → Общий доступ.
4. Разреши приложению доступ к локальной сети, когда iOS спросит.

## Честные оговорки

- Интерфейс и комментарии — по-русски, как и вся Наоми.
- С бесплатным Apple ID подпись живёт 7 дней — потом приложение перестаёт запускаться,
  лечится повторным Run из Xcode за минуту. С платным аккаунтом разработчика — год.
- Работает в одной сети с Маком (домашний Wi-Fi). Доступ из большого мира — через
  [Tailscale](https://tailscale.com): поставь его на Мак и айфон и впиши tailscale-адрес
  Мака в настройках приложения.
- Красивое размытие нижнего края — API iOS 26; на iOS 17–18 приложение работает,
  но без стеклянных эффектов.
- Фото, голос и пуш-уведомления пока не умеет — это следующие главы.
